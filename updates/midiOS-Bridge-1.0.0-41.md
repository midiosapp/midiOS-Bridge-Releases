# midiOS Bridge 1.0.0 (41)

- Fixes an issue where Ableton Live could show incoming MIDI activity while MIDI notes did not reach tracks.
- Adds valid CoreMIDI host timestamps to all MIDI sent through the Bridge virtual input.
- Requires macOS 15.6 or later.
