# midiOS Bridge 1.0.0 (36)

- Adds a first-run Setup Assistant that guides new users through installing Bridge in Applications and completing initial setup.
- Provides automatic Ableton Integration guidance, including installation and the required `midiOS8x4`, `midiOS_IN`, and `midiOS_OUT` selections in Ableton Live.
- Improves the DMG installation experience with a professional Retina presentation and a clear drag-to-Applications workflow.
- Runs as a menu-bar application for access to Bridge controls and status.
- Provides MIDI routing and local-network transport between midiOS and connected systems.
- Installs and manages the bundled Ableton Integration in the external Ableton User Library under `Remote Scripts/midiOS8x4`.
- Uses signed Sparkle updates for future Bridge releases.
- Requires macOS 15.6 or later.
